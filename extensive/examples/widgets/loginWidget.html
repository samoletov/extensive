Login Widget example

This is a most simple example of a widget generated with the extensive framework. It contains two different views, a LoginView and a RegisterView. Both are set using the setView method of the widget.

After pressing the 'Login' button, the credentials are delegated to the LoginController. This controller verifies the credentials (try homer@simpsons.com,duff) and if they are valid an authenticated event is fired by the widget.

If the 'Register' button is pressed, the LoginController stores the email in the application model and the RegisterView retrieves it from there.
